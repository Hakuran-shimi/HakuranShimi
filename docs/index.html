PROFILE

清水　陵真　(しみず　りょうま)

2005年生まれ。中学時代は、ソフトテニス部に所属。部内で一番上手だったにもかかわらず、「練習が好きなのであって、試合となると話は違う。」と言い、選手権大会辞退の経歴を持つ。 その後、高校、大学も同じように自己との闘いに熱を注ぐ。特技は側転。趣味はナンプレ。伸ばしたいところは、クロールをもっと早く泳ぐこと。
